Browser and Device

This website is based on Bootstrap framework which supports several desktop and mobile browsers. However, old browsers might not display our website correctly.

Supported browsers

Supported browsers and devices are shown below. This table is taken from the official Bootstrap website.

Chrome Firefox Internet Explorer Opera Safari
Android Supported Not Supported N/A Not Supported N/A
iOS Supported N/A Not Supported Supported
Mac OS X Supported Supported Supported Supported
Windows Supported Supported Supported Supported Not Supported

Note that Internet Explorer 6 or lower are not supported. For Linux platform, the website should look and behave well enough in Chrome and Firefox.

Recommendation

Although our website can run on mobile devices, it is recommended that you browse our website using latest versions of web browsers shown above from your desktop PC with screen resolution not less than 970 px for the best display.
