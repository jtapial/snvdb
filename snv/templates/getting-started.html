3 Steps to View

Viewing information can be very easy within only 3 simple steps!

Search

The first step is to retrieve what you are actually interested in from all the information stored in the database. Depending on your needs, SNVdb offers three search modes: by Uniprot accession number, by SNV FTID or by disease name.

Just select your preferred search mode from the drop-down menu in the home page, and enter your query in the search bar below.

Select

If your search returns more than one entry from the database, you will be taken to the Search Results page. Simply click on your favourite entry from the list.

In case you mistyped something, you will be able to see the search term that you used and type the right word in.

Visualize

Finally, you can see your entry here. Depending on whether it is a protein, a SNV, a disease, an interaction or an interface, the page will look slightly different:

Understanding Uniprot entries

An Uniprot entry describes a protein in the database. It is defined by the ID number by which you can find the protein in Uniprot.

These entries are divided in five sections:

  • Basic information
  • Overall graphics
  • Annotations
  • Interactions
  • SNVs

Basic information

Here you can find some useful data about the protein and the gene encoding for it (including links to NCBI/Gene and Uniprot) as well as the link to the raw sequence (with an option to export it as a FASTA file).

If the protein has any associated interactions, you can also find an image of the 3D structure of the protein.

Also, you can toggle each of the other sections on and off by clicking on the tick boxes in the "Display" box.

Overall graphics

Whilst the SNVs map to defined sequence positions for a given protein, the interface residues for each of its interacting partners vary.

In this section, you can see both things at the same time on a graph. The "Default" tab shows only the SNVs.

The top, blue rectangle represents the protein sequence, with the SNVs plotted in red, green, blue or purple (depending on their association with disease).

The grey rectangle shows the Pfam domains detected in the protein sequence, highlighting them in red and green, alternatively. Hover over each domain to see what kind of domain it is, along with some data associated with it.

To add the information about interface residues in the interaction with a particular partner, just go to the top of this section and click on the corresponding tab. The interface residues of the protein for a that partner will be highlighted in yellow.

Finally, hover over a highlighted residue to see some more information about it. If you click on a residue with a SNV, you will go to the SNV entry describing it.

All this information is also plotted residue by residue in the detailed alignment below.

Annotations

This section describes the Pfam domains that have been detected for the protein. The domain positions are plotted with respect to the protein sequence, and you can also see a list with the positions where the sequence matches the Pfam HMM for that domain, along with the statistical parameters that validate the domain.

If you want to find out more about that domain, and other proteins sharing the same domain, you can click on the external link to Pfam, where you will find more information about domains.

Interactions

Here we include a list of binary interactions for the protein of interest. For each one, you can see its ID on our database, the two proteins taking part in the interaction, and whether the structure of the complex has been experimentally determined or computationally modelled.

Under this information, the site displays a representation of the two proteins, with the interface residues marked in yellow.

There are also buttons that will lead you to the interaction entry (where you can see the 3D structure of the complex via JsMOL), the interface entry (where you can see the atom-atom interactions that hold the complex together) and the PDB file describing it.

If you prefer to have the PDB file to visualize it your preferred viewer (such as PyMOL), you can also download it from here.

SNVs

Here you can find a list of SNVs that have an effect on the protein sequence. For each one, we show the position, the change on the sequence, the FT id and the rs id (so that you can find it in Uniprot and dbSNP).

If the SNV is associated with a disease, you can also see which disease it is, and the OMIM code describing it.

To find out more about each SNV, just click on the FT id link and we will take you to the SNV entry describing it on our database. We also provide some links to external databases (OMIM and dbSNP). If so many links seem a bit confusing, just hover on each link to see if it links to one of the entries in SNVdb, or it leads to an external database.

Understanding SNV entries

A SNV entry describes a single nucleotide variation affecting a particular protein. These entries are divided in four sections:

  • Basic information
  • Disorders associated to the SNV
  • Mapping mutated SNV to the sequence
  • Other SNVs sharing the same position

Basic information

Here we provide some data about the SNV, such as the protein and position that it maps to, the change in sequence that it causes, and whether it is associated to any disease. If the SNV has a dbSNP id (rs ID), you can also see it here, and find a link to the dbSNP entry.

Disorders associated to the SNV

This is a list of the diseases directly associated to this variation. You can find the link to the SNVdb disease entry and the OMIM code and link for each disease.

Mapping mutated SNV to the sequence

Here you can see the SNV in the protein sequence context. The residue corresponding to the SNV is highlighted.

Other SNVs sharing the same position

If there are more SNVs mapping to this position in the protein, you can find a list here, along with some data and the link to the SNVdb entry for each SNV.

Understanding disease entries

A disease entry displays information about a particular disorder. These entries are divided in four sections:

  • Basic information
  • Related diseases summary
  • Related SNVs list
  • Related Uniprots

Basic information

Here we display the disease ID (which is the same as the OMIM ID) and the name of the disease.

Related diseases summary

Here you can find diseases sharing some SNVs with the disease of interest. Just click on the links and see the disease entries for them.

Related Related SNVs list

This is a list of the SNVs associated with this disease. Click on the link on the "ID" column and see the SNV entries for them.

Related proteins and their associated diseases

Some proteins are associated with more than one disease. If any of the proteins associated with the disease of interest are associated with other disorders, you will see them in this list.

Browser and Device

This website is based on Bootstrap framework which supports several desktop and mobile browsers. However, old browsers might not display our website correctly.

Recommendation

Although our website can run on mobile devices, it is reccommended that you browse our website using latest versions of web browsers shown above from your desktop PC with screen resolution not less than 970 px for the best display.
